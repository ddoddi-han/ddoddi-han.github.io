2022.03 - 현재
  • 2명 규모로 약 2개월 동안 V2 계열 Page Router 프론트엔드를 Next.js App Router 기반 V3 구조로 재구축하고, route group, layout/provider, TanStack Query, 검색·테이블·모달·폼 패턴을 표준화했습니다.

  • 2024-07-10 기준 상 6건·중 4건이던 웹 취약점 결과를 2024-08-05 기준 상 0건·중 0건으로 낮췄습니다.
  • 태그 관련 시계열 데이터를 Redis 기반 조회에서 InfluxDB 기반 조회로 전환하고, 제품 버전 상승에 맞춰 DB migration을 작성했습니다.
